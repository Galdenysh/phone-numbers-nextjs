# Приложение Phone Numbers NextJS

## О проекте

Реактивное приложение Phone Numbers NextJS позволяет вводить телефонные номера и выводить список введенных номеров. Отображается актуальный список на всех устройствах, где было открыто приложение.

## Технологии

Проект написан с использование технологий:

- NextJS
- Redux Toolkit
- Redis
- Socket.IO

## Начало работы

Для запуска приложения в режиме разработчика введите:

```
npm run dev
```

Для просмотра откройте: [http://localhost:3000](http://localhost:3000).
